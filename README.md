# Library Management System

A simple Library Management System developed in Python using file handling.

## Features

- Add new books
- View all books
- Search books
- Delete books
- Store book records in a text file

## Technologies Used

- Python
- File Handling

## How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Run the Python file.
4. Follow the on-screen instructions.

## Project Purpose

This project was created to practice Python programming concepts such as:
- Functions
- Loops
- Conditional Statements
- File Handling
